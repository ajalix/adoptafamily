# Reference Files

This directory contains reference files for the Adopt A Family event. Below is a summary of each file:

## Files

1. **1772060368760_eerntc_2026-2-13_Catholic_Charities_Christmas_Connections_Blank_Spreadsheet_for_Sponors__1_.xlsx**
   - **Description**: Blank spreadsheet template for sponsors. This spreadsheet is used by sponsors to record information about the families they are adopting and the gifts they plan to provide.

2. **1772060369685_pgwzzk_2026-2-13_Catholic_Charities_Christmas_Connections_2026_Christmas_Form_for_Staff__1_.xlsx**
   - **Description**: Form for staff to collect information during the intake process. This form is used to gather details about families in need, including their wish lists and demographic information.

3. **1772060436771_ix5dzm_Registrations_as_of_10.28.25_at_11_am_Deidentified.xlsx**
   - **Description**: Spreadsheet containing registrations as of October 28, 2025. This file includes de-identified data about families and sponsors who have registered for the event.

4. **1772060437498_0wp8k4_Adopt_a_Family_Master_Spreadsheet_2025_Deidentified.xlsx**
   - **Description**: Master spreadsheet for the Adopt A Family event. This file contains comprehensive de-identified data about families, sponsors, and the matching process.

5. **1772060438915_rt54qr_2025-AAF-INSTRUCTIONS.doc.pdf**
   - **Description**: Instructions for participants and organizers of the Adopt A Family event. This document outlines the steps and guidelines to follow during the event.

6. **1772060439549_nl6hdn_2025-AAF-RECEIPT.doc.pdf**
   - **Description**: Receipt template for donations and sponsorships. This document is used to acknowledge receipt of gifts or monetary contributions from sponsors.

7. **1772060440162_hw4ud4_2025-Adopt-A-Family-Flyer.doc.pdf**
   - **Description**: Flyer for promoting the Adopt A Family event. This document can be used to spread awareness and encourage participation in the event.

8. **1772060440794_88f9ey_Email_for_package_email_2025.docx.pdf**
   - **Description**: Email template for sending package information to sponsors. This document is used to communicate details about the families and their wish lists to the sponsors.

9. **1772060441290_91nbii_MIS703.pdf**
   - **Description**: Additional reference material for the Adopt A Family event. This document may contain supplementary information or guidelines relevant to the event.
